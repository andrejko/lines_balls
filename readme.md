# Ping pong game based on [Phaser](http://phaser.io/) engine

![lines-balls demo](assets/lines.png)

## Controls
### General
 * <b>SPACE</b> - launch ball
### Player1
 * <b>Q</b> - up
 * <b>A</b> - down
 * <b>Z</b> - use bonus
 * <b>X</b> - launch bonus
 
### Player1
 * <b>↑</b> - up
 * <b>↓</b> - down
 * <b>M</b> - use bonus
 * <b>N</b> - launch bonus
